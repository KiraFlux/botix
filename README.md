# Botix

Образовательный мобильный робот для 5-11 классов.

Проект разработан для использования в школе при подготовке к олимпиадам по робототехнике (ВсОШ, МОШ, РРО). 
Конструкция спроектирована для производства на стандартном 3D-принтере (FFF), все электронные компоненты доступны и недороги.

## Используемое ПО

| Наименование | Назначение            |
| ------------ | --------------------- |
| FreeCAD      | САПР механики         |
| KiCAD        | САПР электроники      |
| OrcaSlicer   | Слайсер для 3D-печати |

## Структура репозитория

| Путь              | Назначение                                                        | Комментарий                                                                   |
| ----------------- | ----------------------------------------------------------------- | ----------------------------------------------------------------------------- |
| `botix.fcstd`     | Главная сборка в FreeCAD                                          | Все механические детали                                                       |
| `import/`         | Временные STEP-файлы деталей робота, импортированные из КОМПАС-3D | Используются в сборке `botix.fcstd` как внешние ссылки до завершения миграции |
| `models/`         | 3D-модели покупных компонентов (моторы, платы, датчики)           | Используются как внешние ссылки в сборке, не являются печатаемыми деталями    |
| `artifacts/`      | Генерируемые файлы (STEP, 3MF, рендеры)                           | Не хранятся в репозитории, доступны в [Releases](../../releases)              |
| `docs/`           | Техническая документация                                          | Руководство по сборке, список деталей, настройки печати, электрические схемы  |
| `firmware/`       | Исходный код прошивок                                             | Представлены PlatformIO проектами                                             |
| `scripts/`        | Скрипты автоматизации                                             | Экспорт STEP, рендеринг, подготовка 3MF для OrcaSlicer                        |
| `boards/`         | Исходники печатных плат в KiCad                                   |                                                                               |
| `print_profiles/` | Профили для OrcaSlicer                                            | Настройки принтеров                                                           |

## Использование

1. Для получения готовых файлов для печати перейдите в раздел [Releases](../../releases) и скачайте архив с артефактами. Откройте файлы `.3mf` или импортируйте STEP в OrcaSlicer.
2. Для модификации конструкции:
   - Установите FreeCAD.
   - Откройте файл `botix.fcstd`.
   - Внесите изменения.

Подробная инструкция по сборке и настройке приведена в [docs/assembly_guide.md](docs/assembly_guide.md).
