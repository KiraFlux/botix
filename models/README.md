# Модели покупных компонентов

Этот каталог содержит 3D-модели (STEP) стандартных электронных и механических компонентов, которые используются в роботе, но не печатаются на 3D-принтере.

Эти модели являются постоянной частью репозитория и используются в сборке `botix.fcstd` как внешние ссылки. Они обеспечивают корректное отображение посадочных мест и проверку совместимости деталей.

## Структура

| Каталог      | Назначение                                        |
| ------------ | ------------------------------------------------- |
| `devboards/` | Платы управления (Arduino, ESP и т.п.)            |
| `motors/`    | Двигатели и сервоприводы                          |
| `power/`     | Блоки питания, преобразователи напряжения         |
| `sensors/`   | Датчики (ультразвуковые, ИК, TOF, IMU)            |
| `misc/`      | Прочие компоненты (муфты, колёса, макетные платы) |

## Авторство и атрибуция

Некоторые модели в этом каталоге взяты из открытых библиотек (GrabCAD, 3D ContentCentral и др.) или созданы сторонними авторами. Если внутри файла присутствует информация об авторе, она сохранена.

Если вы обнаружили здесь вашу модель и не нашли упоминания вашего авторства, пожалуйста, сообщите об этом через [Issues](../../issues). Ваше авторство будет добавлено в соответствующий файл или в общую таблицу атрибуции.
