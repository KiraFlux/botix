# Визуальные материалы

Этот каталог содержит изображения, рендеры и фотографии, используемые в документации проекта.

## Структура

Файлы именуются по шаблону: `{объект}_{тип}_{номер}.{расширение}`

| Тип          | Назначение                  |
| ------------ | --------------------------- |
| `render`     | Рендеры из FreeCAD          |
| `photo`      | Фотографии реального робота |
| `screenshot` | Скриншоты из документации   |
| `diagram`    | Схемы и чертежи             |
