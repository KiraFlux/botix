# Временные STEP-файлы печатаемых деталей

Этот каталог содержит STEP-модели всех механических деталей робота, экспортированные из КОМПАС-3D.

На текущий момент эти файлы **используются в сборке `botix.fcstd`** как внешние ссылки. Это временное решение на период миграции проекта с проприетарного САПР на FreeCAD.

По мере того как оригинальные детали будут перепроектированы заново внутри `botix.fcstd` (с использованием параметрического моделирования и внешних ссылок на компоненты из `models/`), соответствующие STEP-файлы из этой папки перестанут использоваться. После завершения миграции вся папка `import/` будет удалена.

## Структура

| Каталог        | Назначение             |
| -------------- | ---------------------- |
| `chassis/`     | Детали шасси и корпуса |
| `manipulator/` | Детали манипулятора    |
| `fasteners/`   | Втулки и крепёж        |
| `mounts/`      | Крепления для датчиков |
